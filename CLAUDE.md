# Claude Code 행동 강령

## 에이전트 및 리서치

- 웹 탐색, 리서치, 서브에이전트 활용(opus4.6 포함)은 **임의 판단으로 진행** 후 보고만 하면 됨. 별도 허락 불필요.
- **복잡한 의사결정, 설계, 계획**은 반드시 **opus4.6 서브에이전트**에게 요청.

## 프로젝트 경계

- E:\game의 다른 프로젝트는 참고 가능하나 파일 직접 의존(import) 엄금.
- 다른 프로젝트 코드가 필요한 경우: 해당 파일을**복제하거나 신규 생성**할 것.

## 대량 구현 전 조사
대량 구현 전에는 반드시 코드베이스 전반을 읽어서 프로젝트 전반을 파악할 것.
이미 알고 있는 경우 생략하나, 단편적인 정보만 지녔을 경우엔 총체적인 지식 습득을 위해 전반적으로 파악할 것.

## 대량 구현 후 검수

대량 구현 완료 후에는 반드시 **opus4.6 서브에이전트**로 다음 항목을 검토하고 수정 시도:

- 코드 내 모순 (인터페이스 불일치, 타입 오류 등)
- 명백한 로직 오류
- 알려진 문제점 및 버그
- 실제 사용 패턴에서 부자연스러운 케이스
- 예약어·내장 식별자와의 이름 충돌
